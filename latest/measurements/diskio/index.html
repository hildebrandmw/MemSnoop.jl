    Disk IO

    Disk IO

    MemSnoop.DiskIO — Type.

    Record the rchar, wchar, read_bytes, and write_bytes fields of /proc/pid/io.

    Each measurement returns a NamedTuple with names rchar, wchar, readbytes, and writebytes.

    Example read from /proc/pid/io:

    rchar: 3089822
wchar: 139463
syscr: 159
syscw: 178
read_bytes: 0
write_bytes: 4096
cancelled_write_bytes: 0

    becomes

    (rchar = 3089822, wchar = 139463, readbytes = 0, writebytes = 4096)
